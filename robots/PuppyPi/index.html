PuppyPi

PuppyPi is equipped with RPi 4B and utilizes ROS and Moth server.
Therefore, it is possible to control it using the ROS node and the WebSocket protocol.
By using the Moth server, users can control it through real-time video using the camera.
Furthermore, PuppyPi can show various performances.

CONTROL KEY

⬆️ : FORWARD
⬇️ : BACKWARD
⬅️ : LEFT
➡️ : RIGHT
Number 1: stand
Number 2 : liedown
Number 3 : lookdown
Number 4 : sit
